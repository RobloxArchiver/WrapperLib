# Getting GetJob()
`GetJob()` is also parented to `GetRemotes()` so to use it you'd once again needed. 

```lua
local GetJob = GetRemotes:GetJob();
```

## GiveCustomJob(id)

```lua
GetJob:GiveCustomJob("string");
```

| Arguments | Usage                               | Default |
| --------- | ----------------------------------- | ------- |
| id        | Sets the ID used. Must be a string. | N/A     |
